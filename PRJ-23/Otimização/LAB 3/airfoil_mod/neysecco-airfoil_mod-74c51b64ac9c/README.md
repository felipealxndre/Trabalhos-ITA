# airfoil_mod

**airfoil_mod** is a Python module that facilitates the generation and manipulation of airfoil sections.

## Features

The following parametrizations are currently implemented:

- 4-digit NACA airfoils
- 5-digit NACA airfoils
- CST parametrization
- Strechinsky's parametrization

## Prerequisites

- Python 3
- Numpy
- Scipy
- Matplotlib

## Installation

Open a terminal instance at the repository root folder and run a *pip* install:

```sh
pip install .
```

If you are making changes to the source code, you may also install it with the editable option (-e).
Then any modifications are directly applied without needing to reinstall.

```sh
pip install -e .
```
